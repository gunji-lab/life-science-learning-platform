# My Lab β1

東洋大学生命科学部の研究室を掲載した、研究室紹介Webアプリのβ1です。現在は生命科学科13研究室・生物資源学科13研究室を登録しています。

## 構成

```text
index.html
styles.css
app.js
data/labs.json
data/departments.json
data/events.json
docs/
assets/
```

学科情報は `data/departments.json` で管理しています。学科名、表示順、説明文、学科カラーを保持し、研究室一覧・学科フィルタ・クイックインデックス・受験生向けページに反映します。

研究室情報は `data/labs.json` で管理しています。研究室名、教員名、Question、紹介文、研究方法、大区分、キーワード、おすすめ、授業、外部リンク、公式記事リンク、出典、確認メモを保持しています。

オープンキャンパスなどの受験生向けイベント情報は `data/events.json` で管理しています。イベント名、日程、プログラム、時間、場所、対象、担当研究室を研究室データとは分けて保持しています。

学科追加の手順は `docs/adding-a-department.md`、JSON項目の整理は `docs/data-schema.md` にまとめています。

## 確認方法

JSONを読み込むため、ファイルを直接開くのではなくローカルサーバーで確認してください。

```bash
python3 -m http.server 8123
```

その後、ブラウザで `http://127.0.0.1:8123/` を開きます。

## β1メモ

- 公式研究室一覧をもとに、現在は26研究室を登録しています。
- 公式研究室紹介ページとWeb体験授業リンクを中心に構成しています。
- LINK@TOYOなど東洋大学公式サイト内の記事・インタビューが見つかった教員には、`media_links` として公式記事リンクを追加しています。
- 「この先生から学べる授業」は2026年度公式シラバスの教員名検索から抽出し、原則として座学2件＋実験1件、共通導入科目を避ける方針で3件ずつ掲載しています。
- researchmap、担当授業の主担当／共同担当、年度変更の可能性は `needs_review` を残しています。
- KAKENリンクはUIに表示していません。
- 講義カードには個別リンクを付けず、公式シラバスへのリンクをセクション末尾に1つ置いています。
- 興味検索は、JSONの `keywords`・`methods`・紹介文などの事実データから、アプリ側の変換ルールで高校生向け入口タグへ自動分類しています。
- タグ設計は `major_categories` と `keywords` に分けています。植物・微生物・動物などの大区分を先に付け、研究内容キーワードは5〜10個程度に絞っています。
- 詳細キーワード検索では、`キリン → 哺乳類 → 動物` のような上位タグをアプリ側で自動補完しています。
- 「受験生のみなさんへ」ページを追加し、2026年8月21日・22日のオープンキャンパス情報を日付ごとに切り替えて表示しています。イベント情報は研究室JSONとは別ファイルなので、次回イベント時は `data/events.json` の差し替えで更新できます。
- 学科名・学科説明・学科カラーは `data/departments.json` に分離しました。次に学科を追加するときは、研究室データの `department` と学科設定の `name` を一致させれば、フィルタや表示順に自動反映されます。

## Lab Card Refresh

- 研究室一覧は、まず「研究室名から見る」「おすすめから見る」「研究の問いから見る」の3つの入口を選ぶ構成に整理しました。
- 研究室名インデックスには、研究室名、教員名、キーワード3件を表示します。
- おすすめインデックスには、研究室名、教員名、「こんな人におすすめ」を表示します。
- 研究の問いインデックスには、研究室名、教員名、Question、キーワード3件を表示します。
- CTA文言を「次の扉をひらく →」に変更しました。
- 研究室一覧トップでは詳細カード一覧を常時表示せず、各入口で学科を選んだ後にインデックスを表示します。
- 興味検索の結果はカードではなくミニマムなクイックインデックスで表示し、クリックで詳細を開く形にしました。
- 各インデックスをクリックすると詳細モーダルを開きます。
- 学科カラーは通常時に強く出さず、hover / focus / tap時に控えめに出るようにしました。
- カード全体クリック、Enter / Space操作、お気に入りボタン、検索時のマッチ表示、詳細モーダルの動作は維持しています。
