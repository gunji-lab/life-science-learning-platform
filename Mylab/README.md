# My Lab β1

東洋大学生命科学部の生命科学科13研究室・生物資源学科13研究室を掲載した、研究室紹介Webアプリのβ1です。

## 構成

```text
index.html
styles.css
app.js
data/labs.json
assets/
```

研究室情報は `data/labs.json` で管理しています。研究室名、教員名、Question、紹介文、研究方法、キーワード、おすすめ、授業、外部リンク、出典、確認メモを保持しています。

## 確認方法

JSONを読み込むため、ファイルを直接開くのではなくローカルサーバーで確認してください。

```bash
python3 -m http.server 8123
```

その後、ブラウザで `http://127.0.0.1:8123/` を開きます。

## β1メモ

- 公式研究室一覧をもとに、26研究室を登録しています。
- 公式研究室紹介ページとWeb体験授業リンクを中心に構成しています。
- researchmapと担当授業は、確認できたもの以外は `needs_review` を残しています。
- KAKENリンクはUIに表示していません。
- 講義カードには個別リンクを付けず、公式シラバスへのリンクをセクション末尾に1つ置いています。
